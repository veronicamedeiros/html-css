Código Fonte

Código fonte são comandos, partes de código.

O comando document.getElementaryById('teste') é escrito em linguagem JavaScript. Na frase anteiror, ficou ruim de ler, mas há um comando que deixa todas as letras ocupando um mesmo tamanho de espaço, a diferebça a seguir: document.getElementaryById('teste'). É chamada de fonte monoespaçada e usa-se a tag CODE, é ótma para usar em códigos.
